GATT Server

GATT (Generic Attribute Profile) is the protocol BLE uses to exchange data. Connecting opens a channel to the remote GATT server running on the device.

3

Services

A GATT Service is a collection of related data points (e.g., "Heart Rate Service", "Battery Service"). Each service has a unique UUID.

Primary Services

    4

    Characteristics

    A Characteristic is a single data value within a service. Each characteristic has a UUID and a set of properties: READ, WRITE, and/or NOTIFY.

    Characteristics

      5

      Read / Write / Notify

      Operations you can perform on the selected characteristic, depending on its advertised properties.

      READ WRITE NOTIFY

      READ Fetch the current value of the characteristic.

      WRITE Send a value to the device. Type a string or hex bytes below.

      Prefix values with 0x for hex bytes, otherwise the text is sent as UTF-8.

      NOTIFY The device will push updates automatically — no polling needed.
